# uni-app 目录结构

```jsx
pages 业务页面文件存放的目录
static 静态文件目录(images之类)
App.vue App 全局应用配置

main.js 初始化入口文件
manifest.json 多端配置信息
pages.json 配置页面路由、导航等信息，类似原生小程序的 app.json
```
