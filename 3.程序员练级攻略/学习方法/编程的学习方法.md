## 学习技巧

1. 要先学会一门语言，语言都是想通的，把一门语言学透了，再去学习其他语言，就很容易触类旁通；
2. 坚持在实践中学习，知行合一，不要只是看课程听音频，一定要动手去实现一遍，在实践中学习
   1. 学习一门技术的关键在于动手实践，不管你看了多少课程和资料，如果不动手实践，知识终归不是你自己的
   2. 踏实，积少成多
   3. 集中在一个项目中学习，大的项目开发
   4. 学习的渐进性
3. 软件开发是一个综合的过程，除了编程语言，组件，接口，还有设计模式和设计思想
   1. 将业务逻辑充分分析，编写出可行的业务逻辑代码，也是开发中最重要和需要锻炼的

当用户在微信的存储空间界面中点击缓存清理按键里时，是会删除之前缓存的小程序吗

官方说小程序的缓存与它的代码包是同进同退，只有代码包清理了，小程序缓存才会被清理。
而代码包，在微信的存储空间里，无论是清理“聊天记录”、还是清理“缓存”，都对它产生不了影响。
所以结论应该是：微信自带的存储空间清理，对小程序构成不了威胁。
有网友做过评测，只有重装微信，小程序代码包才会被清理：https://post.smzdm.com/p/aqndnrkp/





之前碰到 map 组件在滑动的时候，没办法跟着一起滑动应该就是因为 map 为原生组件，其层级在一般组件之上。

vue也是基于 web Component标准实现的

vue将 template模板里面的每个节点都解析为一个 web component node节点

即使是一个简单的变量文本，也解析为一个独立的节点；

好处：在数据更新时，更新的是节点的文本；小程序的实现机制与vue是一样的

不一样的是：

小程序的原生组件永远是放在视图组件上面的

将解析后生成的原生组件放在一般组件的上面
