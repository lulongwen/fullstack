# eslint



## eslint配置 



The href attribute is required for an anchor to be keyboard accessible. Provide a valid, navigable address as the href value.

package.json 配置 eslintConfig rule

```jsx
"eslintConfig": {
  "extends": "react-app",
  "rules":{
    "jsx-a11y/anchor-is-valid":"off"
  }
},
```
